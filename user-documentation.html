EduTrack LMS Documentation

Complete user guide for the Education Management System

🎯
What is EduTrack LMS?

EduTrack LMS is a comprehensive Learning Management System designed for educational institutions. It provides tools for managing schools, classes, students, courses, trainers, and generating detailed report cards with performance tracking.

Key Features

  • School Management - Create and manage multiple schools with branches
  • Class Management - Organize classes with assigned trainers and courses
  • Student Management - Enroll students, track attendance and performance
  • Course Management - Define courses with duration and grade levels
  • Report Cards - Generate comprehensive report cards with custom templates
  • Attendance Tracking - Daily attendance with session-wise logs
  • Multi-Role Access - Admin, Manager, Trainer, and Student roles

System Requirements

Supported Browsers

  • Google Chrome (latest version)
  • Mozilla Firefox (latest version)
  • Microsoft Edge (latest version)
  • Safari (latest version)

Internet Connection

Stable internet connection required for real-time data synchronization.

System Access

How to Access the System

  1. Open your web browser

    Launch Google Chrome, Firefox, Edge, or Safari on your computer or mobile device.

  2. Enter the system URL

    Type the following address in your browser's address bar:

    http://43.205.116.187/app/
  3. Login with your credentials

    Use the username and password provided by your administrator. See the User Credentials section for default accounts.

  4. Access your dashboard

    After successful login, you'll be redirected to your role-specific dashboard.

⚠️ Important Note

Make sure to use HTTP (not HTTPS) when accessing the system. The server is configured for HTTP access only.

Role-Based Access

Administrator

Admins have complete control over the system with access to:

  • Create and manage schools
  • Create managers and assign them to schools
  • Create trainers and students
  • Manage all classes and courses
  • Configure report card templates
  • View system-wide analytics
  • Access recycle bin for deleted items

Manager

Managers are assigned to specific schools and can:

  • Manage their assigned school's data
  • Create and manage classes
  • Create trainers and students within their school
  • View school-level reports
  • Manage academic sessions and terms

Trainer

Trainers are assigned to specific classes and can:

  • View their assigned classes
  • Take attendance for sessions
  • Enter student performance data
  • Generate report cards for their classes
  • View student profiles

Student

Students have limited access to view:

  • Their own profile information
  • Class details and assigned courses
  • Attendance records
  • Performance metrics
  • Generated report cards

Administrator Guide

Getting Started as Admin

As an administrator, you have full control over the EduTrack LMS. This guide will walk you through the essential setup tasks.

1. Creating Schools

  1. Navigate to Schools

    From the admin dashboard, click on "School Management" in the left sidebar.

  2. Add New School

    Click the "Add School" button and fill in the required information:

    • School Name
    • Address
    • Contact Information
    • School Code (unique identifier)
  3. Save School

    Click "Create" to add the school to the system.

2. Creating Managers

  1. Navigate to "User Management" → "Managers"
  2. Click "Add Manager"
  3. Fill in manager details (Name, Email, Phone)
  4. Set username and password
  5. Assign to one or more schools
  6. Click "Create Manager"

3. Setting Up Academic Years

  1. Go to "Academic Settings" → "Sessions & Terms"
  2. Create a new academic session (e.g., "2025-2026")
  3. Set start and end dates
  4. Add terms/semesters within the session
  5. Mark one session as "Active"

4. Creating Courses

  1. Navigate to "Course Management"
  2. Click "Add Course"
  3. Enter course details:
  • Course Code (unique)
  • Course Title
  • Description
  • Duration (in weeks)
  • Grade Level (optional)

5. Managing Report Card Templates

Creating Parameter Sets

  1. Go to "Report Card Settings" → "Parameter Sets"
  2. Create a new parameter set with metrics (e.g., Attendance, Homework, Class Participation)
  3. Define metric keys with weightage
  4. Save the parameter set

Creating Templates

  1. Navigate to "Report Card Templates"
  2. Click "Create Template"
  3. Select a parameter set
  4. Configure layout and grading scale
  5. Save as draft or publish
✅ Best Practice

Set up your academic year and courses before creating classes. This ensures proper assignment of courses to classes.

Manager Guide

Managing Your School

As a manager, you are responsible for day-to-day operations of your assigned school(s). Here's how to manage your responsibilities effectively.

1. Creating Classes

  1. Navigate to Classes

    From your dashboard, click "Class Management" in the sidebar.

  2. Create New Class

    Click "Add Class" and provide:

    • Class Name (e.g., "Grade 5-A")
    • Select Academic Session
    • Select Academic Term
    • Assign a Trainer (or leave unassigned)
    • Select Courses to include
  3. Save Class

    Review and click "Create Class"

2. Managing Trainers

  1. Go to "User Management" → "Trainers"
  2. Click "Add Trainer"
  3. Fill in personal details
  4. Create login credentials
  5. Assign to specific school(s)
  6. Optionally assign to specific class(es)

3. Managing Students

Adding Individual Students

  1. Navigate to "Student Management"
  2. Click "Add Student"
  3. Fill in student details (Name, DOB, Parent Info)
  4. Assign to a class
  5. Create login credentials

Bulk Import

  1. Go to "Import" → "Students"
  2. Download the CSV template
  3. Fill in student data following the template format
  4. Upload the CSV file
  5. Review and confirm import

4. Managing Academic Sessions

  1. Navigate to "Academic Settings"
  2. View available sessions created by admin
  3. Create terms for your school within the session
  4. Set term start and end dates
  5. Mark current term as active

5. Viewing Reports

Access the "Reports" section to view:

  • School-wide attendance summary
  • Class performance analytics
  • Trainer activity reports
  • Student progress reports

Trainer Guide

Your Trainer Dashboard

As a trainer, you manage daily class activities including attendance, student performance, and report card generation.

1. Viewing Assigned Classes

  1. Login to the system
  2. Your dashboard displays all assigned classes
  3. Click on a class card to view details
  4. View student list, courses, and schedules

2. Taking Attendance

Creating a Session

  1. Navigate to your class
  2. Click "Create Session" or "Take Attendance"
  3. Select the date and session type
  4. Enter session notes (optional)

Marking Attendance

  1. You'll see a list of all students in the class
  2. Mark each student as:
    • Present - Student attended
    • Absent - Student did not attend
    • Late - Student arrived late
    • Excused - Absence with valid reason
  3. Click "Save Attendance" when complete

3. Entering Performance Data

  1. Go to class details page
  2. Click on "Performance" or "Enter Grades"
  3. Select the metric (e.g., Homework, Class Participation)
  4. Enter scores for each student
  5. Add remarks if needed
  6. Save the performance data

4. Generating Report Cards

  1. Navigate to "Report Cards" section
  2. Select the class and term
  3. Choose a report card template
  4. Review auto-calculated metrics
  5. Add trainer remarks
  6. Generate individual or batch report cards
  7. Download as PDF or print

5. Viewing Student Profiles

  1. Click on any student name in your class
  2. View their complete profile:
  • Personal information
  • Attendance history
  • Performance across all metrics
  • Previous report cards
  • Parent/guardian contact details
💡 Tip

Take attendance regularly and enter performance data consistently. This ensures accurate and comprehensive report cards at the end of each term.

Student Guide

Accessing Your Information

As a student, you can view your academic records, attendance, and report cards through the EduTrack LMS portal.

1. Logging In

  1. Visit the system URL provided by your school
  2. Enter your username (usually your email or student ID)
  3. Enter your password
  4. Click "Sign In"

2. Your Dashboard

After login, you'll see:

  • My Class - Your current class information
  • Assigned Trainer - Your class teacher's details
  • Courses - Subjects you're studying
  • Attendance Summary - Your attendance percentage
  • Recent Report Cards - Latest term results

3. Viewing Report Cards

  1. Click on "Report Cards" in the sidebar
  2. View list of all generated report cards
  3. Click on any report card to view details
  4. Download as PDF for your records
📄

Report Card View

Shows grades, attendance, and trainer remarks

4. Checking Attendance

  1. Navigate to "My Attendance"
  2. View attendance by month or term
  3. See present/absent/late counts
  4. View detailed session logs

5. Updating Profile

  1. Click on "My Profile"
  2. View your personal information
  3. Contact your trainer if any details need correction
🔒 Important

Keep your login credentials secure. Do not share your password with anyone. If you suspect unauthorized access, contact your trainer immediately.

Each school record contains:

  • Address - Physical location

  • Principal/Head - Contact person details
  • Branches - Multiple locations if applicable

School Settings

Academic Configuration

  • Set academic session format (e.g., 2025-2026)
  • Define term structure (Semesters/Trimesters)
  • Configure grading scales
  • Set attendance policies

Managing Branches

  1. Go to School Details
  2. Click "Manage Branches"
  3. Add branch locations with addresses
  4. Assign managers to specific branches

Creating and Managing Classes

Class Creation Checklist

  • ✅ School is created and configured

  • ✅ Academic term is defined

Updating Class Information

Report Card System

Comprehensive Student Evaluation

The report card system allows trainers to generate detailed performance reports combining attendance data, course metrics, and qualitative assessments.

Understanding Report Cards

Components

  • Student Information - Name, Class, Roll Number
  • Academic Details - Session, Term, School
  • Attendance Summary - Present, Absent, Late counts with percentage
  • Course Performance - Grades/Scores for each subject
  • Metrics - Custom evaluation criteria (Homework, Participation, etc.)
  • Remarks - Trainer comments and recommendations

Setting Up Report Cards

Step 1: Create Parameter Sets (Admin)

  1. Go to "Report Card Settings" → "Parameter Sets"
  2. Click "Create Parameter Set"
  3. Name your parameter set
  4. Add metrics (e.g., Attendance, Homework, Behavior)
  5. Define metric keys with scoring criteria
  6. Save the parameter set

Step 2: Create Templates (Admin)

  1. Navigate to "Report Card Templates"
  2. Click "Create Template"
  3. Select a parameter set
  4. Configure layout and appearance
  5. Set grading scale
  6. Save and publish

Step 3: Configure Class Report Cards (Manager/Admin)

  1. Open class details
  2. Go to "Report Card Configuration"
  3. Select a template
  4. Select or customize parameter set
  5. Save configuration

Generating Report Cards

  1. Go to "Report Cards" section
  2. Select class and term
  3. Review auto-populated data:
    • Attendance is automatically calculated
    • Course grades are pulled from performance records
  4. Add or edit metric scores
  5. Enter trainer remarks
  6. Preview the report card
  7. Generate for single student or entire class
  8. Download as PDF or print
✅ Pro Tip

Set up your report card configuration at the beginning of each term. This ensures consistent evaluation throughout the academic period.

Attendance Management

Tracking Student Attendance

The attendance system allows trainers to record daily attendance, view historical data, and generate attendance reports.

Attendance Status Types

Status Description Impact
● Present Student attended the session Positive count
● Absent Student did not attend Negative count
● Late Student arrived late Partial count
● Excused Absence with valid reason Neutral count

Taking Attendance

For Trainers

  1. Login and navigate to your class
  2. Click "Create Session" or "Take Attendance"
  3. Select the date (defaults to today)
  4. Choose session type (Morning/Afternoon/Full Day)
  5. Mark each student's status
  6. Add notes if needed
  7. Save the attendance record

Bulk Operations

  • Mark All Present - Quick option for full attendance days
  • Copy from Previous - Copy yesterday's attendance as starting point

Viewing Attendance Reports

Class-Level View (Manager/Trainer)

  • Daily attendance sheets
  • Monthly summary with percentages
  • Student-wise attendance history
  • Absence pattern analysis

Student-Level View (Student)

  • Personal attendance calendar
  • Month-wise statistics
  • Overall attendance percentage
  • Session details

Attendance in Report Cards

Attendance data automatically appears in report cards showing:

  • Total working days in the term
  • Days present
  • Days absent
  • Days late
  • Attendance percentage

Troubleshooting

Common Issues & Solutions

Login Issues

Cannot Access the System

Problem: Page keeps reloading or shows "Not Secure" warning

Solution:

  1. Ensure you're using http:// (not https://)
  2. Clear browser cache (Ctrl+Shift+Delete)
  3. Try accessing in incognito/private mode
  4. Check if URL is correct: http://43.205.116.187/app/

Invalid Credentials

Problem: "Invalid username or password" error

Solution:

  1. Check caps lock is off
  2. Verify username (usually email format)
  3. Contact admin if password forgotten

Data Issues

Cannot Create Class

Problem: "Academic year not found" or similar

Solution: Ensure academic session and terms are created first by admin/manager

Students Not Appearing

Problem: Added students not showing in class

Solution:

  1. Verify students are assigned to the correct class
  2. Check if students are marked as "Active"
  3. Refresh the page after adding students

Report Card Issues

Template Not Available

Problem: No templates showing when generating report cards

Solution: Contact admin to create and publish a report card template

Attendance Data Missing

Problem: Attendance shows 0% in report card

Solution: Ensure attendance has been recorded for the selected term

Performance Issues

Page Loading Slowly

  • Check internet connection stability
  • Clear browser cache
  • Try a different browser
  • Close other tabs/applications using bandwidth
